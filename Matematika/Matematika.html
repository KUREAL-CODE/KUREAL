      n disebut bilangan pangkat(Eksponen)

      Xn=X.X.X.X.X.X...


      (-2)3=(-8)

      Jika bilangan pangkat adalah ganjil, maka hasil perpangkatan pasti minus (-)

      Jika bilangan pangkat adalah genap, maka hasil perpangkatan pasti plus (+)

      Sifat Sifat Eksponen

      Sifat pertama : am x an = am + n


      Contoh :

      23X24=23+4

      27


      35X32=35+2

      37


      X3.X9=X3+9

      X12

      Bilangan pokok(basis) harus sama, tidak boleh beda

      Sifat kedua : am : an = am - n


      Contoh :

      23:24=23-4

      2-1


      35:32=35-2

      33


      X3:X9=X3-9

      X-6

      Bilangan pokok(basis) harus sama, tidak boleh beda

      Sifat ketiga : (am)n=amxn

      Sifat keempat : amxbm=(axb)m

      Sifat kelima : am:bm=(a/b)m

      Sifat keenam : a-m=(1/am)

      Sifat ketujuh : an/m=anakar m
